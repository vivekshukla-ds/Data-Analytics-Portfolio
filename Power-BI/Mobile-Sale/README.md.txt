# Mobile Sales Analysis – Power BI Dashboard

## Project Overview
This Power BI dashboard analyzes mobile sales data to identify trends,
performance metrics, and business insights across products, regions,
and time periods.

## Key KPIs
- Total Sales
- Total Quantity Sold
- Revenue by Brand
- Sales by Region
- Monthly Sales Trend

## Tools Used
- Power BI
- DAX (for calculated measures)

## Visual Design
This report uses custom visual elements such as icons and background
designs applied directly within the Power BI report to improve
readability and user experience.

## File Information
- `Mobile_Sale.pbix` – Interactive Power BI dashboard file

## Notes
Due to GitHub limitations, `.pbix` files cannot be previewed online.
The report can be explored by downloading and opening it in Power BI Desktop.
