\# Budget vs Forecast Analysis – Power BI



\## Project Overview

This Power BI project analyzes Budget vs Forecast performance across

organizational units to identify variances, utilization trends, and

operational insights.



The dashboard is designed for management-level decision making and

provides both high-level summaries and detailed drill-down views.



\## Key Dashboards

\- Executive Summary

\- Branch Management

\- Division Management

\- Employee Details

\- Hours Analysis



\## Datasets Used

Multiple datasets were integrated and transformed using Power Query:

\- `budget-and-forecast.xlsx`

\- `datatrain.xlsx`

\- `hours.xlsx`

\- `people-and-tasks.xlsx`



\## Key Metrics \& Insights

\- Budget vs Forecast variance analysis

\- Resource utilization and hours tracking

\- Employee-level performance visibility

\- Branch and division-wise comparisons



\## Tools \& Techniques

\- Power BI Desktop

\- Power Query for data cleaning and transformation

\- DAX measures for calculated KPIs

\- Custom Power BI theme for consistent visual design



\## Visual Design

This report uses a custom Power BI theme (`PowerBI.json`) along with

designed visual layouts to improve readability and presentation quality.
