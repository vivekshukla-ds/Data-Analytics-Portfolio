# IMDB Movie Rating Scraper

## Problem Statement
Scrape top-rated movies data from IMDB for analysis purposes.

## Data Source
- Website: https://www.imdb.com/chart/top/
- Data is collected dynamically using web scraping.

## Tools & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas

## Approach
1. Send request to IMDB Top Movies page
2. Parse HTML using BeautifulSoup
3. Extract movie title, year, and rating
4. Save the data into CSV format

## Output
- imdb_movies.csv (generated in outputs folder)
